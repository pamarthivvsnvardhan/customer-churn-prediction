# Customer Churn Prediction Using Machine Learning

This project uses Machine Learning techniques to predict whether a telecom customer is likely to churn or stay with the company. The model helps businesses identify at-risk customers and improve customer retention strategies.

---

## Project Overview

Customer churn prediction is an important problem in the telecom industry. In this project, I analyzed customer data, performed data preprocessing, visualized insights, and built a predictive machine learning model to classify customers as:

- Churn
- Not Churn

The project demonstrates the complete Machine Learning workflow from data analysis to model evaluation.
